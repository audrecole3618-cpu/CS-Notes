# Intro to Computer Science
CS is about **problem-solving** using logic.

### Core Concepts:
* **Algorithms:** Step-by-step logic to solve a problem.
* **Data Structures:** How we organize data (Arrays, Lists).
* **Binary:** Computers speak in 1s and 0s.
---

### ⚡ The Core Pillars of Hardware

Every modern system relies on four fundamental hardware blocks working together to process, store, and move data securely:

* **The CPU (Central Processing Unit):** The actual "brain" of the machine. It executes instructions and runs calculations using billions of microscopic switches.
* **RAM (Random Access Memory):** The short-term, volatile scratchpad. The CPU uses this to hold files and programs you are actively running *right now*. Turn off the power, and RAM completely wipes clean.
* **Storage (SSD / Hard Drive):** The long-term, non-volatile memory vault. This is where your operating system, photos, and files live permanently even when the power is completely cut.
* **Motherboard:** The central nervous system. A massive circuit board that physically connects all components together so they can talk to each other at lightning speed.
