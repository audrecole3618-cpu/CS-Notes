# Linux & Terminal Survival Kit
On a Mac, your terminal is a Gateway to the OS.

### Essential Commands
* `ls -la`: Show ALL files, including hidden ones (like your `_sidebar.md`).
* `mkdir [name]`: Create a new folder.
* `touch [name].md`: Create a new blank file instantly.
* `grep`: Searching for text inside files. (Essential for looking through logs!)
* `chmod`: Changing file permissions. (Who can read/write your notes?)
